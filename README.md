## Requirements
iOS 6+
## Installation

TTKeyboardAvoidingScrollView is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "TTKeyboardAvoidingScrollView"
```
